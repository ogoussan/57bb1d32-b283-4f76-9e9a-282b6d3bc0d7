## Setup

### Prerequisites:

- npm

### Clone the Project

```sh
$ git clone https://github.com/ogoussan/57bb1d32-b283-4f76-9e9a-282b6d3bc0d7
```

### Install dependencies

```sh 
$ cd ./events-app
$ npm install
```

### Run app in development mode

```sg
$ npm start
```

Open [http://localhost:4200](http://localhost:4200) to view it in the browser.
